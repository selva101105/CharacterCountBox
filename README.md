 Character Counter Message Box

Description

This project is a simple **Character Counter Message Box** built using **HTML, CSS, and JavaScript**.
It allows users to type a message and see the **real-time character count**.

Spaces are **not counted** as characters.
The maximum allowed characters is **200**.

Features
* Real-time character counting while typing
* Spaces are ignored in character count
* Maximum limit of **200 characters**
* Warning message shown when limit is reached
* Prevents typing beyond the character limit
* Simple and clean UI

Technologies Used
* HTML
* CSS
* JavaScript

How It Works
* User types text in the textarea
* JavaScript removes spaces before counting characters
* The counter updates live (example: 120 / 200 characters)
* When the limit is reached, a warning message appears
* Extra characters are automatically blocked
